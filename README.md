# Smart Hostel Complaint & Maintenance Manager

A Java OOP based console project designed for hostels, PGs, and student accommodations where students can register complaints such as electricity issues, water leakage, fan repair, WiFi problems, etc. Admin can manage and resolve complaints efficiently.

--- 

##  Project Overview

Managing hostel maintenance manually is slow and inefficient. This project digitizes the complaint system using Java Object-Oriented Programming concepts.

Students can:

- Raise maintenance complaints
- View complaint details
- Track complaint status

Admin can:

- View all complaints
- Resolve complaints
- Update complaint status

🧠 OOP Concepts Implemented

✅ Encapsulation
- Data members inside classes with controlled access.

✅ Inheritance
- Student extends User
- Admin extends User
- ElectricComplaint extends Complaint
- PlumbingComplaint extends Complaint

✅ Polymorphism
- Method overriding used in complaint display methods.

✅ Constructors
- Used for object initialization.

✅ Abstraction
- Complaint system represented through class structure.


▶️ How to Run Project

Step 2: Run Main Class
